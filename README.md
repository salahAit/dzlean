# 📖 منصة DzLearn التعليمية

> **منصة تعليمية للمنهج الجزائري**  
> _Algerian Educational Curriculum Platform_

منصة "DzLearn" هي منصة تعليمية سريعة وحديثة، مبنية كبديل مفتوح وخفيف لمواقع الفروض والاختبارات الجزائرية (مثل DzExams)، مخصصة لعرض الفروض، والاختبارات، والدروس لجميع المراحل (ابتدائي، متوسط، ثانوي).

## 🚀 التشغيل السريع (Quick Start)

```bash
# 1. تثبيت التبعيات | Install dependencies
bun install

# 2. إنشاء قاعدة البيانات وإضافة البيانات الأولية | Seed Database
# سيقوم هذا السكربت بإنشاء content.db وإضافة نموذج للمدارس والمواد والفروض
bun run scripts/seed-education.ts

# 3. تشغيل خادم التطوير | Start Dev Server
bun run dev
```

## 🛠️ التقنيات (Tech Stack)

تم بناء المشروع لتوفير أداء فائق وسرعة في عرض المحتوى:

| المكون        | التقنية                  | السبب                               |
| ------------- | ------------------------ | ----------------------------------- |
| **Runtime**   | **Bun**                  | أسرع بأضعاف من Node.js              |
| **Framework** | **SvelteKit (Svelte 5)** | أداء عالي وسهولة تطوير (Full-Stack) |
| **Database**  | **SQLite**               | قاعدة بيانات مدمجة وسريعة جداً      |
| **ORM**       | **Drizzle**              | Type-safe وخفيفة                    |
| **CSS**       | **TailwindCSS v4**       | تصميم متجاوب ومباشر                 |

## 📂 هيكل المشروع (Structure)

```
dzlearn/
├── src/
│   ├── lib/
│   │   └── server/db/    # إعدادات Drizzle وجداول قواعد البيانات
│   ├── routes/           # Pages (File-based routing):
│   │   ├── [level]/      # صفحة السنوات (مثل: /moyen)
│   │   │   └── [year]/   # صفحة المواد (مثل: /moyen/1am)
│   │   │       └── [subject]/ # صفحة الفروض (مثل: /moyen/1am/math)
│   └── app.html & app.css # التصميم الأساسي
├── data/
│   ├── content.db        # المحتوى التعليمي للمنهاج الجزائري
│   └── users.db          # بيانات المستخدمين (للتوسعات القادمة)
└── scripts/
    └── seed-education.ts # سكربت ملء نماذج البيانات
```

## 🚀 النشر (Deployment)

لرفع التطبيق على خادم VPS، يمكنك استخدام السكربت المرفق:

```bash
./deploy.sh full    # الإعداد الأولي والرفع
./deploy.sh deploy  # لنشر تحديثات النظام
```
